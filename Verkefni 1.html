Upphaf

Í þessu verkefni sem fjallað er um nánar hér rétt að neðan er tilgangurinn að búa til heimasíðu og kynna hópinn sem vinnur að lokaverkefninu og hvers eðlis það er. Varð fyrir valinu að búa til hátalara þeas gjörðina utan um hann og má sjá nánar um það verkefni undir Lokaverkefni. Hópurinn samanstendur af mér, Eðvarð Þór, Birgi og Viktori.

Lýsing á verkefni

Verkefni þetta fjallar um að gera heimasíðu, þessa tilteknu síðu sem þú ert núna að skoða. Til að byrja með var stofnaður aðgangur á Github. Þegar það hafði verið gert var stofnað repository og benti github mér á síðuna https://html5up.net/ til þess að ná í template sem var og er mjög þægilegt. Í fyrstu notaði ég bara github online en fljótlega var ég farinn að sjá að betra væri að nota Visual Studio. Í því forrita ég allt og skrifa sem ég í kjölfarið krem(push) svo inná síðuna mína í gegnum Git bash sem treður(push) þessu inní repository á vefnum. Ég mæli virkilega með þessu þar sem ég er mjög fatlaður í svona löguðu en þarna hefur allt umhverfi verið sniðið að þörfum okkar. Gallinn við þessi tilbúnu templete er kannski að þau eru stútfull af allskonar Demo sem ég amk þurfti ekki og tók talsverðan tíma að eyða, breyta og sníða að mínum þörfum. Mæli samt hiklaust með.

Nánar um notkun git bash

Þegar þú ert búinna að hlaða niður git bash þá er hann mjög auðveldur í notkun. Farið er í möppuna þar sem öll heimasíðan er geymd, þar er síðan hægri smellt og kemur upp valmöguleikinn Git bash here, við það opnast glugginn sem unnið er með. Við sérhverjabreytingu í visual studio eða eftir 1 dag, hvað sem menn velja eru svo 4 skref sem ég kýs að nota í git bash.

1. git status - sérð það sem þú ætlar að breyta

2. git add . - keyrir allt saman í gegn.

3. git commit -m"messege"

4. git push - treður öllu saman inná svæðið þitt online.
